# MyFontCraft

自分だけのオリジナルフォントを作れるWebサービス。

## 特徴

- **かんたん**: テンプレートを印刷して手書き → スキャンするだけ
- **プライバシー安心**: すべての処理がブラウザ内で完結。画像がサーバーに送られません
- **無料**: ユーザー登録不要、完全無料

## 使い方

### 1. テンプレートをダウンロード

「テンプレート生成」からPDFをダウンロードして印刷します（約80ページ）。

### 2. 手書きで記入

各マスに文字を書きます。

- 1文字につき2マスあります
- 最低1つ書けばOK
- 上手く書けた方に✓をつけてください
- 失敗したら×をつけてください

### 3. スキャン or 撮影

- スキャナーでスキャン、または
- スマホで撮影

斜めに撮っても自動補正されます。事前の画像加工は不要です。

### 4. アップロード

画像をZIPまたはフォルダでアップロードします。

### 5. ダウンロード

処理が完了すると、フォントファイル（.ttf）をダウンロードできます。

## 対応文字

- ひらがな
- カタカナ
- 常用漢字（2,136文字）
- 英数字・記号

## 動作環境

モダンブラウザ（Chrome, Firefox, Safari, Edge）
