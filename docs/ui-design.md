# UI/UXデザイン

## デザインコンセプト

**スキューモーフィズム（紙のイメージ）+ ドット絵風アクセント**

- 背景: クリーム色の紙テクスチャ
- カード: 少し浮いた紙のような影
- ボタン: 押せそうな立体感
- フォント: 手書き風または明朝体
- アイコン・装飾: ドット絵風

---

## ロゴデザイン

### コンセプト

**マインクラフト風（パクリにならない程度）**

- タイトル「MyFontCraft」をピクセル/ブロック風フォントで表現
- 立体感のある文字（斜め影付き）
- 色: 茶色〜ベージュ系（紙・クラフト感）
- Minecraftの土ブロックではなく「紙・原稿用紙」モチーフ
