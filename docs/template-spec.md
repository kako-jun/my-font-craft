├─────────────────────────────────────────────────────────────────┤
│▓★                                                            ☆▓│
│▓                                                              ▒│

## 1文字あたりの配置

```

┌──┐ ┌──────┐┌──────┐
│見│ │ 文字 ││ 文字 │   見本: 10mm × 15mm
│本│ │ 記入 ││ 記入 │   マス: 15mm × 15mm × 2
│字│ │ 欄① ││ 欄② │   マス間: 2mm
└──┘ ├──────┤├──────┤
     │check ││check │   チェック欄: 3mm
     └──────┘└──────┘

見本(10mm) + 間隔(2mm) + マス(15mm) + 間隔(2mm) + マス(15mm) = 44mm
+ 列間余白(3mm) = 47mm
```

## 左右縦グレースケールバー

ページの左右余白に縦方向に配置。スキャン時の影勾配（上下・左右）を補正するための基準。

### 配置

| 項目  | 左バー                    | 右バー |
| ----- | ------------------------- | ------ |
| X座標 | 2mm                       | 203mm  |
| Y開始 | 17mm（上マーカーと同じ）  | 17mm   |
| Y終了 | 272mm（下マーカーと同じ） | 272mm  |
| 幅    | 5mm                       | 5mm    |
| 全長  | 255mm                     | 255mm  |

### ステップ

10段階、各 5mm幅 × 25.5mm高さ（= 255mm / 10）。上から下へ黒→白のグラデーション。

| 段階 | 濃度 | RGB     |
| ---- | ---- | ------- |
| 1    | 100% | #000000 |
| 2    | 90%  | #1A1A1A |
| 3    | 80%  | #333333 |
| 4    | 70%  | #4D4D4D |
| 5    | 60%  | #666666 |
| 6    | 50%  | #808080 |
| 7    | 40%  | #999999 |
| 8    | 30%  | #B3B3B3 |
| 9    | 20%  | #CCCCCC |
| 10   | 10%  | #E6E6E6 |

### 用途

スキャン時に左右のバーを読み取り、各ステップの実測値と期待値の差分を計算。
ページ全体のピクセルに対して上下（Y方向）・左右（X方向）の線形補間で影勾配を補正する。

## QRコード仕様

### 技術仕様

| 項目               | 値                                   |
| ------------------ | ------------------------------------ |
| サイズ             | 8mm × 8mm                            |
| QRコードバージョン | Version 3〜5（データ量に応じて自動） |
| エラー訂正レベル   | M（15%復元可能）                     |
| 最大データ量       | 約130バイト                          |

### 内容（JSON）

```json
{
  "p": "mfc",
  "v": 2,
  "pg": 3,
  "t": 51,
  "m": 2
}
```

| キー | 意味                                 |
| ---- | ------------------------------------ |
| p    | project識別子（"mfc" = MyFontCraft） |
| v    | version                              |
| pg   | page番号                             |
| t    | total pages                          |
| m    | マス数/文字                          |

**NOTE**: 文字リスト（`c`）はQRに含めない。��ージ番号（`pg`）から `getCharactersForPage(pg - 1)` で導出する��漢字30文字のJSON化がQRの容量上限（約130バイト）を超えるため。

### 配置

右下マーカー (x=192, y=272) の左隣に配置。マーカー内側に収まり、2mmの間隔を確保。

```
         QR(8mm)    マーカー(8mm)
x:  182──────190  192──────200
y:  272──────280  272──────280
```

スキャン時は下部20%領域（y=237.6mm〜297mm）を探索するため、y=272mmのQRは確実に検出される。

### 不採用とした要素

| 要素                         | 理由                           |
| ---------------------------- | ------------------------------ |
| スケールマーク               | 外枠サイズ（15mm）で正規化可能 |
| バーコード（フォールバック） | QRコードで十分、インク節約     |

## 採用ルール

### マークの意味

| チェック欄のマーク | 動作                 |
| ------------------ | -------------------- |
| ✓（チェック）      | これを採用           |
| ×（バツ）          | これは除外           |
| 空欄               | デフォルトルール適用 |

### デフォルト動作

- チェックも×もない場合 → 一番右の記入済みマスを採用
- 複数に✓がある場合 → 複数バリエーションとして採用

### 例

```
┌──┐┌──┐
│あ││あ│ ← 両方記入、マークなし → 右を採用
├──┤├──┤
│  ││  │
└──┘└──┘

┌──┐┌──┐
│あ││あ│ ← 左にチェック → 左を採用
├──┤├──┤
│✓ ││  │
└──┘└──┘

┌──┐┌──┐
│あ││あ│ ← 両方にチェック → 2バリエーション採用
├──┤├──┤
│✓ ││✓ │
└──┘└──┘

┌──┐┌──┐
│あ││  │ ← 1つだけ記入 → それを採用
├──┤├──┤
│  ││  │
└──┘└──┘
```

## テンプレートデザイン

### コンセプト

**ドット絵風でおしゃれに、かつインク節約**

### ドット絵風デザインの定義

| 項目     | 仕様                           |
| -------- | ------------------------------ |
| 角       | 直角のみ、角丸なし             |
| 線幅     | 0.5pt（約0.18mm）              |
| 色       | ベタ塗り（グラデーションなし） |
| フォント | ピクセルフォント風             |

### 各要素

- **外枠**: ドット絵風の枠線（ピクセルっぽい角）
- **四隅マーカー**: ★☆マーク（8mm四方、シンプルな星形）
- **ヘッダー**: ドット絵風の装飾ライン
- **ページ番号**: ピクセルフォント風
- **タイトル「MyFontCraft」**: ロゴと同じドット絵風

### デザイン原則

- **楽しさ**: 単調な作業を少しでも楽しく
- **インク節約**: 塗りつぶしは最小限、線のみで表現
- **視認性**: おしゃれさより機能性優先（書く領域は明確に）

## ヘッダー タイトル＋フォント名

### 表示ロジック

タイトル「MyFontCraft」とフォント名を1行で表示する。フォント名にASCII以外の文字（日本語等）が含まれるかで描画方法を分岐:

- **ASCII のみ**: `MyFontCraft — "FontName"` を Helvetica で一括描画
- **非ASCII含む**: 「MyFontCraft」を Helvetica で描画し、`— "フォント名"` を Canvas→PNG→PDF埋め込みでタイトル直後に配置

フォント名画像の幅は最大80mmに制限。配置X座標はHelveticaのwidthOfTextAtSizeで動的計算。

## ヘッダー雑学コメント

### 概要

ヘッダー2行目に、ページの内容にまつわる短い雑学コメントを表示する（漫画の作者コメント風）。
日本語テキストをCanvas APIで画像化し、pdf-libでPNG埋め込みする方式。

### 表示仕様

| 項目           | 値                                                  |
| -------------- | --------------------------------------------------- |
| 位置           | ヘッダー2行目、右寄せ（x=190mmを右端とする）        |
| フォントサイズ | 6pt相当（画像化）                                   |
| 色             | グレー (#666666)                                    |
| 背景           | 白                                                  |
| 長さ           | 20-40文字程度                                       |
| 右端制限       | x=190mm（右上マーカー x=192 との間に2mm余白を確保） |

### コメント選択ロジック

1. ページ番号に固有の雑学があればそれを優先（例: 42ページ → 「生命、宇宙、そして万物の答え」）
2. 最終ページは完走メッセージ
3. ページの文字種（ひらがな/カタカナ/ASCII/漢字）に応じた雑学を選択
4. どれにも該当しない場合は汎用雑学

### データ管理

- `src/data/trivia.ts` にハードコード
- `getTriviaForPage(pageIndex, pageChars, totalPages)` で取得
- 外部API不使用
