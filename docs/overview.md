# プロジェクト概要

## MyFontCraftとは

手書きテンプレートからオリジナルフォントを作成するWebサービス。

## 特徴

- **ブラウザ完結**: 全処理がブラウザ内で完結
- **プライバシー**: ユーザーデータはサーバーに送信されない
- **無料**: ユーザー登録不要、完全無料

## ワークフロー

```
1. テンプレートPDFをダウンロード・印刷
      ↓
2. ユーザーが手書きで記入
      ↓
3. スキャン or スマホ撮影
      ↓
4. 画像をアップロード（ZIP or フォルダ）
      ↓
5. 自動処理 → フォントファイル(.ttf/.otf)をダウンロード
```

## 設計方針

| 項目 | 方針 |
|------|------|
| ユーザー登録 | なし |
| データベース | なし |
| サーバー処理 | なし（全部ブラウザ） |
| プライバシー | 画像がサーバーに送信されない |
| コスト | ほぼ無料（Cloudflare無料枠） |

## AIの使用方針

| 用途 | AI使用 |
|------|--------|
| 文字の補間・生成 | ❌ **使わない**（本人の字を100%使用） |
| ✓印の検出 | ✅ 使う |
| ×印の検出 | ✅ 使う |
| 空マスの検出 | ✅ 使う |

**重要**: 本人が書いた文字をそのまま使用する。AIで文字を生成・補間することはしない。
